# Icon Placeholder

The extension requires icon files in the `icons/` directory.

## Required Icons:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

1. Go to https://www.canva.com

3. Design with purple theme
